safe_require (...)
Requires a module, returning nil if it doesn’t exist. Note that this differs from the regular require() function, which will raise an error if the given module does not exist.

Parameters:

  • ...: The argument(s) to pass to require()

Returns:

    The module’s table, or nil if it doesn’t exist
