Functions

platform ()
Returns the current execution platform. Values can include "windows", "darwin", "linux", or whatever the lower-case version of `uname` returns. "unknown" is returned if the output from `uname` is empty (e.g. in case of an error).

Returns:

    A string indicating the current execution platform
architecture ()
Returns the current processor architecture. The string returned will be one of 'x86', 'x64', 'ia64', 'unknown', or whatever `uname -p` returns.

Returns:

    A string indicating the processor architecture
